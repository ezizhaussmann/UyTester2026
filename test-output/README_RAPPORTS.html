🚀 Comment Exécuter les Tests

1. Exécuter les tests avec Maven :

mvn clean test -Dsurefire.suiteXmlFiles=test-reports.xml

2. Voir le rapport Extent :

Ouvrir : test-output/ExtentReport.html

3. Générer et voir le rapport Allure :

mvn allure:serve
# ou
mvn allure:report
# puis ouvrir : target/site/allure-maven-plugin/index.html

🔧 Utilisation dans vos Tests

Pour utiliser les rapports dans vos tests existants :

1. Ajouter les annotations :

@Listeners({ReportListener.class})
public class VotreTest {
    @Test
    @Description("Description de votre test")
    @Severity(SeverityLevel.CRITICAL)
    public void votreTest() { ... }
}

2. Initialiser les rapports :

@BeforeSuite
public void setUp() {
    ExtentReportManager.initReports();
}

@AfterSuite
public void tearDown() {
    ExtentReportManager.flushReports();
}
